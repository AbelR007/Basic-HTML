Attributes

Attributes provides additional information about a tag or a function.
For eg. "img" has an attribute named "src", "a" has an attribute named "href"

This "style" attribute inside "p" can colour and style a paragraph.

The "lang" attribute in "html" can declare the language for this web page.

The "title" attribute in "p" can show tips when mouse is hovered across.
