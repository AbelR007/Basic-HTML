Styling can be done to a tag type using the below format

Like thos :
"tagname style="style_property:style_value;"
With angular brackets of course

Ever heard of tomato background styling :)

Properties

  • background-colour
  • color
  • font-family
  • font-size
  • text-align
  • border
  • Any many more...
Many Colour Codes can also be used. For eg, hsla, RGB, hsl and Hex code values
RGB takes four parameters:
Eg : rgb(red,green,blue,alpha)
Hex Code Values are good too. Eg : #ff0000
HSL are cool too. Eg : hsl(0, 100%, 50%)
Btw HSLA or HSL stands for (Hue, Saturation, Lightness, Alpha)

Formatting Text

Formatting elements were designed to display special types of text: 
       - Bold text 
       - Important text 
       - Underlined text 
       - Italic text 
       - Emphasized text 
       - Marked text 
       - Smaller text 
       - Deleted text 
       - Inserted text 
       - Subscript text 
       - Superscript text

Bold & Strong are somewhat same but strong is shown as important text while bold isn't

Emphasis & Italics are also somewhat same but Emphasis is shown with an emphasis on the word its shown for while Italics isn't
