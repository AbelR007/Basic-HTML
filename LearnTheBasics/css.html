CSS in HTML

CSS controls the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used.

* BUT you have to LEARN CSS first.

How to use CSS in HTML ?

There are 3 ways to use CSS in HTML. They are :
  1. 1. Inline
  2. 2. Internal
  3. 3. External

Inline

Inline uses the style attribute inside another tags.
It can be added to tag. Refer my LTB attributes file for more idea.

Internal

You can use style tag itself. And then, you can use CSS as you normally do.
Normally, this leads to a lot of confusion if you are creating a big, better site.

External

You can create a .css file and just link it in the index html file.
